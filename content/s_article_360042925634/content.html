Title
Permitted File Types for Attachments
Article Body

There are a number of tools in Domo that allow you to upload files, including Buzz, Projects and Tasks. In the interest of preventing users from uploading malicious files into Domo accounts, Domo allows a specific set of file types to be uploaded. If you attempt to upload a file not within this list, an error message appears.

This list of allowed file types includes:

Intro

Google Search Ads 360 Connector, formerly DoubleClick Search, is a search management platform that helps agencies and marketers efficiently manage some of the largest search marketing campaigns in the world, across multiple engines and media channels. Use Domo's Google Search Ads 360 connector to create custom searches and get data about account, ad, ad group, ad group target, and so on. To learn more about the DoubleClick Search API, visit their page (https://developers.google.com/doubleclick-search/).

You connect to your Google Search Ads 360 account in the Data Center. This topic discusses the fields and menus that are specific to the Google Search Ads 360 user interface. General information for adding DataSets, setting update schedules, and editing DataSet information is discussed in Adding a DataSet Using a Data Connector.

Prerequisites

To connect to your Google Search Ads 360 account and create a DataSet, you must have the following:

  • The Google API Client Id.

  • The Google API Client Secret. 

You can create a Client ID and Secret by going to the Google API Console.

Follow the below steps to create client ID and Client Secret:

1) enable access to the DoubleClick Search API

 2) register this redirect URI: https://oauth.domo.com/api/data/v1/oauth/providers/doubleclick-search/exchange

Connecting to Your Google Search Ads 360 Account

This section enumerates the options in the Credentials and Details panes in the Google Search Ads 360 Connector page. The components of the other panes in this page, Scheduling and Name & Describe Your DataSet, are universal across most connector types and are discussed in greater length in Adding a DataSet Using a Data Connector.

Credentials Pane

This pane contains fields for entering credentials to connect to your Google Search Ads 360 account. The following table describes what is needed for each field:  

Field

Description

Client ID

Enter your Google API Client ID.

Client Secret

Enter your Google API Client Secret.

For information about obtaining these credentials, see "Prerequisites," above.

Once you have entered valid Google Search Ads 360 account's credentials, you can use the same account any time you go to create a new Google Search Ads 360 connector DataSet. You can manage connector accounts in the Accounts tab in the Data Center. For more information about this tab, see Managing User Accounts for Connectors.

FAQs

What version of the Google Search Ads 360 (Doubleclick Search) API does this connector use?

This connector uses version 0 of the Google Search Ads 360 (Doubleclick Search) API (https://searchAds360.googleapis.com/v0).

 

Are there any API limits that I need to be aware of?

No

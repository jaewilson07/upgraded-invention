Intro

You can add Smart Text to a card title or description while working in Analyzer or from a dashboard. Smart Text references certain data elements and updates in place whenever the underlying data changes. You can use more than one piece of Smart Text at a time, if needed.

Anyone with permission to edit a card or dashboard can add/modify associated Smart Text.

Personalized Data Permissions (PDP) do apply when adding Smart Text to a card title or description. For more information about PDP, see Personalized Data Permissions (PDP). 

The Smart Text options are described below:

  • Date range — Shows dates in text. Ex: this quarter, graphed by week
  • Filters — Displays the filters applied. Ex: order status.
  • Variables — When a variable is present, the selected value of the variable is listed.
  • Segments used — Names the visible segments.
  • DataSet info — Displays information about the DataSet such as the date last updated or owner.
 
Note: You can also add Smart Text to Notebook Cards. See Adding a Notebook Card to learn more.

Add Smart Text in Analyzer

Inside Analyzer, follow these steps to add Smart Text to a card title or description:

  1. Hover over the card title or description to display the  Edit option.

     

  2. Select either option to open the Edit Card Title and Description modal.



To continue, see Configure Smart Text.

Add Smart Text on a Dashboard

  1. Navigate to the dashboard where you want to add Smart Text and select  Settings menu >  Edit Dashboard.



  2. In edit mode, hover over the card where you want to add Smart Text and select Edit Content > Edit card title & description.

The Edit Card Title and Description modal displays. To continue, see Configure Smart Text.

Configure Smart Text

  1. In the modal, select + Smart Text for either the Card title or Description fields to open the Smart Text menu.

     

     

     
    Note: If the card does not have any variables, the Variables option is not available. Add a variable to the card to have the option for Variables Smart Text.

  2. Select the type of Smart Text you want to add and follow the prompts to apply logic. The logic depends on the type of Smart Text. See the table below.

     

     
    Note: Your Smart Text configuration can make your card title or description quite lengthy. We recommend being thoughtful about how you apply Smart Text for usability.

    Smart Text Type

    		Logic

    Date range

    • Date range
    • Graph by
    • Compare to
    • Date range field

     

    Segments used

    • Segment name
    • Segment description

    Learn more about segments.

     

    DataSet Info

    • Name
    • Owner
    • Date last updated

     

    Prefix/Suffix — A prefix or suffix is optional. It is a character or characters that come before or after the Smart Text in your card title or description. 

    For example, as a prefix for Date range, you may enter "— " (em dash and then a space). 

    Use the Preview field in the modal to see how your prefix/suffix will appear.

    Variables

    (x) — When the variable value updates, the Smart Text shows the current value.

    Learn more about variables.

    Default text — Text that displays in card search results and the Card Admin Settings.

    Prefix/Suffix — See above.

    Filters

    When you select the Filters Smart Text, you can choose to display the filters applied or select the box labeled Display any filter. After making your choice, choose Select at the bottom of the filter list.

    • Display any filter — This is the default setting. Choosing to display any filter means that when your card is being viewed, any filters applied to the dashboard or card display in the card title or description (wherever you place your Smart Text). 

     

    • Filters applied — Choosing a filter or filters means that when your card is being viewed and the filter or filters are applied, they are listed in the card title or description (wherever you place your Smart Text). The system default is to show the name of the filter if only one filter is selected or to display Filters(x) where x is the number of filters you have selected.

     

    Default text — This text displays when no filters are applied. Ex: `All` Default text also displays in card search results and in the Card Admin Settings.

    Prefix/Suffix — See above.

    Values shown — You can choose to display all the available values or a specific number. For example, if your Smart Text is configured to `Display any filter` and you choose to show only 1 value, only 1 filter will display in your card title or description, even if there are 5 filters applied to the card.

    The default for Values shown is All. Values that exceed the threshold are shown in a tooltip.



  3. Select Apply to keep the logic, Cancel to discard the logic, or Remove to delete that piece of Smart Text from the card title or description.

  4. Select Save to keep your configured Smart Text and close the modal or Cancel to discard your changes.

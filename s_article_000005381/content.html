About 

Sizmek provides tools to build, serve, analyze and target ads to global audiences across a variety of channels, including mobile and video. Combine your Sizmek data with data from sales, finance, and other departments across your organization so you can maximize your ROI. Get easy-to-use, real-time insights into impressions, clicks, interactions, conversions, viewability, and more. Domo's Sizmek connector brings in data from the Sizmek MDX platform and includes reports for Advertisers, Agencies, Campaigns, Markets, Placements, Sites and more.

You can connect to your Sizmek account in the Data Center. This article explains the fields and menus that are specific to the Sizmek Connector user interface. To add DataSets, set update schedules, and edit DataSet information, see Adding a DataSet Using a Data Connector.

Prerequisites

To connect to your Sizmek account and create a DataSet, you must have the following:

  • Username
  • Password
  • API Key

Connecting to Your Sizmek Account

This section lists the options in the Credentials and Details panes in the Sizmek Connector page. The components of the other panes in this page, Scheduling and Name & Describe Your DataSet, are universal across most connector types and are discussed in great detail in Adding a DataSet Using a Data Connector.

Credentials Pane

This pane contains fields for entering credentials to connect to your Sizmek  account. The following table describes what is needed for each field:  

Field

Description

Username

Your Sizmek username.

Password

Your Sizmek password.

API Key

Your Sizemek API key.

Once you have entered valid Sizmek credentials, you can use the same account to create a new Sizmek DataSet. You can manage connector accounts in the Accounts tab in the Data Center. For more information about this tab, see Managing User Accounts for Connectors.

Details Pane

This pane contains a primary Reports menu, along with various other menus which may or may not appear depending on the report type you select.

Menu

Description

Report

Select the Sizmek report you want to run. The following reports are available:

Report Name Description
AdvertisersRetrieves information about advertisers.
Agency ContactsReturns agency contacts based on the selected agency ID.
All CategoriesRetrieves all the categories defined for a specific account.
Banner SizesRetrieves a list of all predefined banner sizes available in Sizmek.
CampaignsRetrieves information about campaigns.
Creative ShopsReturns a list of creative shops.
CurrenciesRetrieves a list of currencies supported by Sizmek.
MarketsRetrieves a list of markets.
Mobile SitesReturns a list of mobile sites (publishers) only.
OfficesRetrieves a list of offices (countries).
PlacementsReturns information about multiple placements.
Performance AnalyticsRetrieves performance report about campaigns.
SitesRetrieves a list of available sites.
AgencySelect the agency.
CampaignsSelect the campaign(s) you want to analyze
Conversion Attribution ModelSelect the conversion attribution model.
Conversion TagsSelect the conversion tags you want to retrieve.
Cookie Window ClickThe number of days a conversion will be attributed to an ad after a click has occurred. The number of days needs to be between 1 and 90
Cookie Window ImpressionThe number of days a conversion will be attributed to an ad after an impression has occurred. The number of days needs to be between 1 and 90
Customization ParameterThe unique value provided by Sizmek that identifies the custom report associated with a specific integration that you want to retrieve.
Data LevelSelect the data level.
DurationSelect the duration for the report (a Single Date, or a Date Range)
Campaign SelectionSelect how you want to specify campaigns.
Tag BreakdownSelect whether to breakdown the data by conversion tags
